# ExamGrader

## **Overview**
This project processes two types of text files: one containing students' answers and the other with the correct answers. It accurately calculates each student's grade based on the data from these files. Additionally, the project computes and reports the overall class performance, including the range of grades and the class average.

### **Technolgy Required**
This project requires the Java programming language. Additionally, an Integrated Development Environment (IDE) capable of compiling and running Java code is needed.
